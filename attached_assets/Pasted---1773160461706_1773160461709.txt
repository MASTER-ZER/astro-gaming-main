أريد إعادة تصميم طريقة عرض الألعاب وباقات الشحن في الموقع لتكون حديثة جدًا، احترافية، وتفاعلية، بدل العرض التقليدي الحالي (قائمة ألعاب + جدول أسعار). 

مهم جدًا:
- لا تغيّر أي منطق backend أو API أو نظام الطلب.
- لا تكسر أي وظيفة موجودة.
- التعديل يكون فقط على مستوى UI / UX / Layout / Motion.
- يجب الحفاظ على نفس البيانات ونفس تدفق الشراء.

الهدف:
تحويل عرض الألعاب والباقات إلى تجربة تفاعلية حديثة تشبه متاجر الألعاب الكبيرة (PlayStation Store / Apple Store / Steam) مع طابع Gaming Premium.

━━━━━━━━━━━━━━━━━━━
1. إعادة تصميم قسم الألعاب
━━━━━━━━━━━━━━━━━━━

بدل القائمة التقليدية، اعرض الألعاب باستخدام Grid من Gaming Cards كبيرة.

كل لعبة يجب أن تكون Card تحتوي:

- صورة اللعبة كخلفية
- Overlay Gradient داكن
- اسم اللعبة
- عبارة مثل:
  "Instant Top-Up"
- نص:
  "Start From $X"
- زر:
  "View Packages"

التصميم:

- border-radius حديث
- glow خفيف حول الكرت
- hover animation:
  - الكرت يرتفع قليلاً
  - يظهر زر الشراء
  - يظهر glow خفيف

عند hover:

- تكبير بسيط للصور
- ظهور طبقة blur خفيفة
- ظهور CTA

━━━━━━━━━━━━━━━━━━━
2. فتح صفحة اللعبة
━━━━━━━━━━━━━━━━━━━

عند الضغط على لعبة:

افتح صفحة مخصصة للعبة تحتوي:

Hero Section:

- صورة كبيرة للعبة
- اسم اللعبة
- عبارة:
  "Instant Top-Up"
- تأثير parallax خفيف للخلفية

━━━━━━━━━━━━━━━━━━━
3. عرض الباقات (أهم جزء)
━━━━━━━━━━━━━━━━━━━

لا تستخدم جدول أسعار.

استخدم Smart Pricing Cards.

مثال:

Card 1
60 UC
Starter Pack
$1

Card 2
325 UC
Most Popular
$5

Card 3
660 UC
Best Value
$10

المتطلبات:

- الكروت تكون في Grid أو Flex layout
- الكرت الأوسط أكبر قليلاً
- إضافة Badge:

Most Popular
Best Value
Recommended

━━━━━━━━━━━━━━━━━━━
4. التفاعل مع الباقات
━━━━━━━━━━━━━━━━━━━

عند Hover على الباقة:

- ارتفاع الكرت قليلاً
- إضافة Glow
- تكبير الرقم الرئيسي
- ظهور زر "Buy Now"

animation:

- smooth ease-out
- transition سريع

━━━━━━━━━━━━━━━━━━━
5. إضافة Highlight للباقات المهمة
━━━━━━━━━━━━━━━━━━━

الباقة الأكثر شراءً يجب أن:

- تكون أكبر قليلاً
- border مضيء
- badge واضح

مثال:

🔥 Most Popular

━━━━━━━━━━━━━━━━━━━
6. تحسين ترتيب الباقات
━━━━━━━━━━━━━━━━━━━

رتب الباقات بحيث:

الأرخص ← اليسار  
الأفضل قيمة ← الوسط  
الأغلى ← اليمين

الوسط يجب أن يكون الأكثر جذبًا.

━━━━━━━━━━━━━━━━━━━
7. تحسين التجربة البصرية
━━━━━━━━━━━━━━━━━━━

استخدم:

- subtle neon glow
- gradient overlays
- smooth shadows
- glassmorphism خفيف

لكن بدون مبالغة.

━━━━━━━━━━━━━━━━━━━
8. الحركة والانيميشن
━━━━━━━━━━━━━━━━━━━

أضف micro-interactions:

- hover lift
- button press animation
- smooth reveal عند تحميل الصفحة
- stagger animation للباقات

لكن:

- حافظ على الأداء
- لا تستخدم أنيميشن ثقيلة

━━━━━━━━━━━━━━━━━━━
9. تجربة الموبايل
━━━━━━━━━━━━━━━━━━━

على الموبايل:

- الألعاب تظهر في Grid عمودين
- الباقات تظهر stacked
- الكرت الأوسط يظهر أولاً

━━━━━━━━━━━━━━━━━━━
10. الحفاظ على الوظائف
━━━━━━━━━━━━━━━━━━━

لا تغير:

- API calls
- checkout logic
- data models
- backend routes
- auth system

فقط حسّن:

- UI components
- layout
- animations
- visual hierarchy

━━━━━━━━━━━━━━━━━━━
النتيجة المطلوبة
━━━━━━━━━━━━━━━━━━━

قسم الألعاب يجب أن يبدو:

- حديث
- تفاعلي
- يشبه متجر ألعاب حقيقي
- أسهل في اختيار الباقة
- أكثر جذبًا بصريًا
- يزيد احتمالية الشراء

بعد التنفيذ:

1. اشرح التغييرات التي تمت
2. اذكر الملفات التي تم تعديلها
3. تأكد أن جميع الوظائف ما زالت تعمل
