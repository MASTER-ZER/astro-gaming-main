أريدك تعيد تصميم واجهة الموقع بالكامل (كل الصفحات + الداشبورد + أي مودال/فورم/كروت/أزرار/هيدر/فوتر) إلى ستايل iOS Glassmorphism (Frosted Glass) احترافي وحديث، بدون ما تغيّر المنطق البرمجي أو وظائف الموقع.

هدف التغيير

تحويل الواجهة الحالية من شكل “داكن تقليدي” إلى شكل زجاج آيفون: خلفية داكنة أنيقة + طبقات زجاج شبه شفافة + blur حقيقي + حدود ناعمة + إضاءة خفيفة.

التغيير شامل: الصفحة الرئيسية + صفحة المستخدم + صفحة الطلبات + الداشبورد + الجداول + الفورم + كل المكونات.

لا أريد “شكل قريب” أو “تعديلات بسيطة” — أريد UI جديد بالكامل بنفس المحتوى والوظائف.

قواعد تصميم (Design System)

خلفية (Background)

خلفية داكنة متدرجة (Deep navy/charcoal) مع لمسات ضوء ناعمة (soft glow blobs) في الزوايا.

ممنوع خلفية مسطحة مملة.

الزجاج (Glass Cards / Panels)

كل الكروت والبوكسات والبانلز:

background: rgba(255,255,255,0.06 ~ 0.12)

backdrop-filter: blur(14px ~ 22px)

حدود: 1px solid rgba(255,255,255,0.12 ~ 0.22)

ظل ناعم + highlight داخلي خفيف.

استخدم طبقات واضحة: background → glass layer → content.

الألوان (Palette)

Primary accent: أزرق iOS أو Cyan خفيف (بدون فاقع).

Text: أبيض/رمادي فاتح مع تباين ممتاز.

Success/Warning/Error: درجات هادئة مش نيون رخيص.

التايبوجرافي

خطوط واضحة وحديثة، أحجام هرمية صح:

عنوان رئيسي قوي

وصف أصغر

أرقام/إحصائيات واضحة

مسافات (spacing) محسوبة، ممنوع الزحمة.

الأزرار (Buttons)

Primary button: زجاج + تدرج خفيف + hover واضح + ضغط (active) واقعي.

Secondary: زجاج أهدى.

حالات Disabled/Loading واضحة.

ممنوع أزرار مسطحة رخيصة.

المدخلات (Inputs / Forms)

Inputs زجاجية بحدود ناعمة، focus ring أنيق، وأخطاء واضحة.

Labels واضحة (مش placeholder بس).

الجداول والداشبورد

الداشبورد لازم يبقى “Premium iOS panel”.

Cards للإحصائيات (مثل: نجاح/ثقة/دعم…): زجاج + أيقونات بسيطة + أرقام كبيرة.

الجداول تتحول إلى:

Table داخل panel زجاجي

صفوف hover ناعمة

Headers ثابتة ومرتبة

على الموبايل تتحول لبطاقات (responsive).

الأنيميشن

Micro-interactions: hover lift بسيط، fade/slide ناعم، transitions محسوبة.

ممنوع حركات كتير أو بطيئة.

الـ Responsive

موبايل أولاً: كل شيء لازم يبقى ممتع على شاشة صغيرة.

Grid ذكي للكروت في الداشبورد.

الـ Accessibility

تباين ممتاز

Focus states للكيبورد

أحجام خط مقروءة

المطلوب تنفيذه تقنياً

اعمل Design Tokens: CSS variables للألوان، الشفافية، blur، border radius، shadows.

اعمل Component Styles موحدة:

glass-card

glass-panel

glass-button

glass-input

stats-card

glass-table

طبّقهم على كل أجزاء الموقع والداشبورد.

لا تكسر أي layout أو functionality.

لو المشروع React/Tailwind: اعمل config/classes utilities للزجاج + components.

لو HTML/CSS عادي: اعمل ملف CSS جديد منظم (مثلاً glass-ui.css) وبدّل الكلاسات في الصفحات.

تسليم نهائي (Acceptance Criteria)

الواجهة كلها شكلها iOS glass premium.

مفيش جزء “نسيته” في الداشبورد أو أي صفحة.

مفيش عناصر متداخلة/غير مقروءة.

الـ UI consistent بنسبة 100%.

ابدأ بتحليل المكونات الموجودة، وبعدها نفّذ refactor للستايل تدريجياً مع اختبار كل صفحة.
